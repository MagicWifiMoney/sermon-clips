| Thumbnails | Canva Pro ($13/mo) | Photoshop ($22/mo) |

### Video Style Mix
- **40% Screen recordings** — Tool tutorials, software walkthroughs
- **30% Talking head** — Strategy, opinion, educational
- **20% Hybrid** (talking head + screen) — Most engaging format
- **10% B-roll heavy** — Equipment reviews, before/after

### Thumbnail Strategy
- Bright yellow/orange backgrounds (stands out in church niche which is very blue/purple)
- Face + expression in every thumbnail
- Large text (3-5 words max)
- Tool/app logos when relevant

### SEO Optimization Per Video
1. **Title:** Include target keyword in first 60 chars
2. **Description:** 200+ words, keyword in first sentence, links to sermon-clips.com
3. **Tags:** Target keyword + 5-10 variations
4. **Chapters:** Add timestamps (boosts CTR)
5. **Cards/End Screens:** Link to related videos + sermon-clips.com
6. **Pinned Comment:** CTA to try sermon-clips.com

### Content Funnel
```
YouTube Video (awareness)
  → Link in description to sermon-clips.com
  → Free resource/template (email capture)
  → Free trial of sermon-clips.com
```
